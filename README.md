Trabalho de conclusão de curso
"Sistema de gerenciamento energético aplicado a quadros de distribuição"

Composto por dois microcontroladores: PIC16f877a e ESP8266.

----------------------------------------------------------

ROTEIRO DE IMPLEMENTAÇÃO do Firmware PIC16f877a utilizando IDE MPLABX v1.85

 * OK  - Timer0 para contagem do tempo 0,1ms
 * OK  - Comunicação serial TX e RX
 * OK  - Tratar comandos recebidos pela porta serial
 * NOK - Configurar i2c
 * NOK - Ler e salvar hora/data no RTC
 * NOK - Ler sensores de corrente
 * NOK - Ler sensores de tensão
